# 我开发了一个名为 马库斯时报 的新闻网站，每天自动更新AI科技日报，帮助你了解AI的最新进展。

#### 特点：

    - 新闻源：量子位、少数派、Techmeme、Reddit
    - 自动播报： 每天上午 9:00 开始抓取新闻，10-20 分钟内完成收集、撰写与发布。
    - 溯源阅读：每一条新闻都保留了原始链接，方便一键直达深度阅读。
    - 永久存档：通过 GitHub Pages 展示 Web 版，所有历史报纸均可在仓库查阅。
---

最初我编写了一个Skill，通过 Openclaw 部署的私有推送机器人，每天从telegram上为我推送新闻，但是一想这可以让更多人了解到AI进展，所以我制作成了每天自动更新的 Web版。希望马库斯时报能成为你每天早晨的一份萨赫蛋糕。
